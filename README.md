# vim_kernel

A Jupyter kernel for Vim script

![vim_kernel](https://raw.githubusercontent.com/mattn/vim_kernel/master/screenshot.png)

## Installation

```
python install.py
```

If you want to install vim_kernel into user's home directory:

```
python install.py --user
```
